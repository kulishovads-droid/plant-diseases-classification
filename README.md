# plant-diseases-classification
# Plant Disease Classification

## Описание проекта

Проект посвящён распознаванию болезней растений с использованием методов компьютерного зрения и нейронных сетей.

## Цель

Создать систему классификации изображений листьев растений.

## Используемые технологии

* Python
* TensorFlow
* MobileNetV2
* Google Colab

## Структура проекта

* notebook — ноутбук с кодом
* dataset — описание используемого датасета
* model — информация о модели
* images — примеры изображений

## Датасет

PlantVillage Dataset (Kaggle)

## Запуск проекта

Установить зависимости:

pip install -r requirements.txt

Запустить notebook из папки notebook.
